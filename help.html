Documentation of SA Map Editor

This is a short documentation of SA Map editor

When starting you should already see Mecatol Rex and a panel to the right with tiles containing Planets

Use Drag and drop to pull tiles from the panel onto the map

You can use the mouse to drag the map around. Click on an already placed tile and move the mouse while still pressing the left button.

To relocate an already placed tile hold Alt while draging the desired tile.

To delete a tile hold shift and click on the tile you wish to delete.

You can zoom the Map using the scroll whele of your mouse.

You can also zoom the tiles in the panel with the slider at the bottom

Different kinds of tiles can be selected using the Tiles menue at the top. Available are

  • Planets
    The panel open at start that contains all planets except the home systems.
  • Special
    This contains all Empty spaces (including wormholes) and red boarder pices that do not have a plenet.
  • Homesystems
    This contains all Homesystems and the numbered Placeholder systems.
  • Tokens
    This contains tokens like Artefacts distantsun tokens and the SA border tokens.

For every planet tile in the panel there is a text in the lower left corner. This defines the game the tile is in cluded

  • TI3: The Twilight Imperium 3 base game
  • TI3:SE: Shatterd Empier expansion for The Twilight Imperium 3
  • TI3:SotT: Shards of the Thron expansion for The Twilight Imperium 3
  • SA: Shatterd Accension Fan expension. You need to print this out if you want to play phyicly
In the left upper corner of the map is a menue Tilesets. You can open it with a mouse click. It will list all Expensions needed to build the displayed map. (Currently only tiles are checked)
